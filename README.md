### 〔 ACERCA DEL SISTEMA 〕

</div>

**Hatsune Miku Bot** es una herramienta de automatización rica en funciones que trae el encanto de Miku a tu WhatsApp. Construido sobre el núcleo de Baileys, opera en tiempo real con módulos de IA, protección de grupos y entretenimiento.

Desarrollo activo — nuevas funciones desplegadas continuamente.

### 〔 INSTALACIÓN EN TERMUX 〕

</div>

**1 —** Actualiza los paquetes base

```sh
pkg update && pkg upgrade
```

**2 —** Instala Git y Node.js

```sh
pkg install git nodejs
```

**3 —** Clona el repositorio

```sh
git clone https://github.com/Brauliovh3/HATSUNE-MIKU.git
```

**4 —** Entra al directorio

```sh
cd HATSUNE-MIKU
```

**5 —** Instala las dependencias

```sh
npm install
```

**6 —** Inicia el bot

```sh
npm start
```

> ⚡ **Termux recomendado →** [Descargar APK](https://files.catbox.moe/b6z9ml.apk)  
> ⚡ Al iniciar, se generará un código de vinculación para WhatsApp Web.
